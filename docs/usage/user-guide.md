# User Guide

This guide covers the main features and how to use them.

## Getting Started

### Registration

1. Open the application in your browser
2. Click "Sign Up"
3. Fill in your details:
   - Username
   - Email
   - Password
4. Click "Register"
5. Verify your email (if email is configured)

### Login

1. Click "Login"
2. Enter your email and password
3. Click "Login"
4. You'll be redirected to the dashboard

## Features Overview

### Dashboard

The main dashboard shows:

- Account summary
- Recent activity
- Quick actions

### Trading Simulation

Access from the main menu → **Simulation**

#### Running a Simulation

1. Go to Simulation page
2. Configure your parameters:
   - Initial Balance: Starting capital
   - Capital Utilization: % of capital per trade
   - Risk Per Trade: Risk amount per trade
   - Risk/Reward Ratio: Target profit vs risk
   - Win Rate: Expected win percentage
   - Trades Per Day: Number of trades
   - Simulation Days: Duration
   - Fees Per Trade: Broker fees
3. Click "Run Simulation"
4. View results:
   - Summary statistics
   - Daily breakdown
   - Monte Carlo analysis

### Goal Planner

Set and track your financial goals:

1. Navigate to **Goal Planner**
2. Enter:
   - Initial Balance: Current capital
   - Target Balance: Goal amount
   - Deadline (months): Timeframe
3. View:
   - Required monthly return
   - Feasibility assessment
   - Recommendations

### Manual Trade Simulator

Simulate manual trades:

1. Go to **Manual Trading**
2. Add trades with:
   - Entry/Exit prices
   - Position size
   - Result (profit/loss)
3. Track your performance over time

### Trade History

View all your trading activity:

- Filter by date
- Search by type
- Export to CSV

### Community

Connect with other traders:

1. **Explore**: Discover communities
2. **Join**: Become a member
3. **Post**: Share insights
4. **Comment**: Engage with others

### Profile Management

Update your profile:

1. Click your username
2. Edit:
   - Avatar
   - Bio
   - Settings
3. Save changes

## Subscriptions

### Viewing Plans

1. Go to **Subscriptions**
2. View available plans:
   - Basic: Limited features
   - Premium: Full features
   - Platinum: All features + priority

### Upgrading

1. Select a plan
2. Choose billing (Monthly/Yearly)
3. Complete payment via Midtrans
4. Enjoy premium features!

### Managing Subscription

- View current plan
- Change billing cycle
- Cancel subscription

## Settings

### Account Settings

- Change password
- Update email
- Enable/disable notifications

### Trading Settings

- Default risk parameters
- Preferred assets

## Troubleshooting

### Can't Login

- Check your credentials
- Reset password if needed
- Contact admin

### Simulation Not Working

- Check input parameters
- Ensure all fields are filled
- Try different values

### Payment Failed

- Check card details
- Ensure sufficient funds
- Try different payment method
