# AI Trading Coach

## Tip - Your Personal Trading Mentor

**Tip** is your AI-powered trading coach powered by Google Gemini.

## Quick Chat Mode (Default)

- Ask about:
  - Market trends (\"What's BTC doing?\")
  - Crypto analysis
  - Trading strategies
  - Risk management
  - Psychology tips

## Coach Mode (Toggle button 🛠️)

**Activate Coach Mode** to get personalized advice using your:

- Recent 10 trades
- Current account balance
- Position analysis

**Perfect for:**

- Trade review (\"Review my last trades\")
- Strategy adjustment
- Risk assessment
- Performance insights

## Example Prompts

**Quick Chat:**

```
What's the market trend?
Is ETH bullish?
Give me a scalping strategy for SOL
```

**Coach Mode:**

```
Analyze my recent losses
Should I increase my risk?
Is my RR ratio good?
Review my trading performance
```

## Insights

Coach Mode provides **Quick Insights** - bullet points summarizing key advice.

## Requirements

- GEMINI_TRADING_COACH_KEY in .env
- Active Manual Trading session for context

**Pro Tip:** Run some manual trades, toggle Coach Mode, ask \"Review my performance\" for personalized feedback! 📊✨
